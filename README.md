# Day 1 Project – Wireshark Introduction

## Objective
To understand how network traffic flows and how to capture packets using Wireshark.

---

## Tools Used
- Wireshark

---

## What I Did
- Opened Wireshark
- Started packet capture on my network interface
- Observed live network traffic
- Filtered traffic using:
  - DNS
  - HTTP
  - TCP

---

## What I Learned
- Devices communicate through packets
- DNS resolves domain names to IP addresses
- HTTP traffic is readable if not encrypted
- TCP controls data transmission

---

## Key Takeaway
Network traffic can be analyzed to detect activity happening on a system or network.
